Принял решение разбить карточку на 7 компонентов, потому что:
1. Компонент ававтар (будет неоднократно повторяться) включает в себя еще 1 компонент это User. Компонент User это фото и имя, данный компонент можно будет переиспользовать отдельно в личном кабинете (наверху справа).
2. Компоненты кнопок выделил отдельно, чтобы их переиспользовать в дальнейшем в разных компоновках.
